Welcome to PyJanitor

A Python library for clean, efficient data preparation

What is PyJanitor?

PyJanitor is a Python library built on top of Pandas that provides a clean, verb-based API for data cleaning and preprocessing tasks. Originally inspired by the R package "janitor," it has evolved into a powerful tool that implements the method chaining paradigm, allowing data scientists to express data processing workflows in a logical, readable sequence.

PyJanitor simplifies common data cleaning operations such as standardizing column names, handling missing values, and transforming data formats—making data preparation more efficient and less error-prone.

Quick Start

Here's how to quickly get started with PyJanitor's method chaining approach:

Basic Usage Example 
import pandas as pd
import janitor as jn

# Create a sample dataframe
df = pd.DataFrame({
    'a': [1, 2, 3],
    'b': [4, 5, 6],
    'Column with spaces': [7, 8, 9]
})

# Use method chaining to clean and transform data
clean_df = (
    df
    .clean_names()                # Convert column names to snake_case
    .rename_column('a', 'x')      # Rename a column
    .add_column('z', df['a'] * 2) # Add a new column
    .drop_columns('b')            # Remove a column
)

print(clean_df)

Pro Tip

The key advantage of PyJanitor is its method chaining approach, which allows you to perform multiple operations in sequence without creating intermediate variables.

Example Use Cases

Example 1: Cleaning Column Names

Column Cleaning Example 
import pandas as pd
import janitor as jn

# DataFrame with problematic column names
df = pd.DataFrame({
    'First Name': ['John', 'Jane', 'Mike'],
    'Last-Name': ['Smith', 'Doe', 'Johnson'],
    'email@address': ['john@example.com', 'jane@example.com', 'mike@example.com'],
    '% complete': [85, 93, 75]
})

# Clean the column names
cleaned_df = df.clean_names()
print(cleaned_df.columns)
# Output: Index(['first_name', 'last_name', 'email_address', 'percent_complete'], dtype='object')

Example 2: Handling Missing Values

Missing Data Example 
import numpy as np
import pandas as pd
import janitor as jn

# DataFrame with missing values
df = pd.DataFrame({
    'name': ['John', 'Jane', None, 'Mike'],
    'age': [30, np.nan, 25, 40],
    'score': [85, 92, None, 78]
})

# Handle missing values with method chaining
cleaned_df = (
    df
    .remove_empty()          # Remove completely empty rows
    .dropna(subset=['name']) # Remove rows with missing names
    .fillna(value={'age': df['age'].mean(), 'score': 0})
)

print(cleaned_df)
